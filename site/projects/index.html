  • Mammogaphic Mass Severity Prediction

    Mammographies can be followed by a biopsy for further examination.
    However, 70% of these biopsies return a benign result, which makes most of them unnecessary.
    In this analysis, we use the provided dataset consisting of the BI-RADS assessment, three BI-RADS attributes and the age of the patient to fit a model for predicting the severity (benign/malignant) of each instance in order to reduce the number of unnecessary biopsies.
